# Stack-Of-Wheat-Seeds-SpeedrunIGT-Category

how to use it:

1. go to C:\Users\[YOUR USERNAME]\speedrunigt\categories (should be there if youve used speedrunigt before)
2. drop the json in there and boom you have it in game under timer category

```
{
    "id": "stack_of_wheat_seeds",
    "name": "Stack Of Wheat Seeds",
    "version": "*",
    "src_category": "",
    "conditions": [
		[
			{
                "type": "player_stat",
                "name": "stack_of_wheat_seeds",
                "category": "minecraft:picked_up",
                "stat": "minecraft:wheat_seeds",
                "goal": 64
            }
		]
	]
}
```
